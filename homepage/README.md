# Homepage

![Dev][dev-image]

## Installation
- Copy `webapps/ROOT/homepage folder` into webapps/ROOT folder
- Copy `webapps/aboutme.war` into webapps folder
- Run Tomcat server

## Tests
- Aceess [the Test page][homepage]

## Development environment
- openJDK 1.8
- Tomcat 8.5
- Amazon Linux
